KAOS Documentation ​

This documentation provides comprehensive coverage of the KAOS, a framework for deploying and managing AI agents with tool access and multi-agent coordination on Kubernetes.

  • Quick Start Guide - Deploy your first agent in minutes
  • Installation - Operator installation and prerequisites
  • Concepts - Core concepts and architecture overview

  • Overview - Framework architecture and design principles
  • Agent - Agent class, configuration, and lifecycle
  • Agentic Loop - Reasoning loop, tool calling, delegation
  • Memory - Session management and event storage
  • MCP Tools - Model Context Protocol integration
  • ModelAPI - LLM client for OpenAI-compatible servers
  • Server - HTTP endpoints and deployment

  • Overview - Operator architecture and controllers

  • ModelAPI CRD - Model API deployment modes
  • MCPServer CRD - MCP tool server configuration
  • Environment Variables - Configuration reference

Reference ​

  • Python API - Python class and method reference
  • Environment Variables - All configuration options
  • Troubleshooting - Common issues and solutions

Development ​

  • Contributing - Development workflow
  • Architecture Decisions - Design rationale
